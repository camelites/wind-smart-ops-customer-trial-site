风电智慧运维管理平台 PC 试用包

生成时间：2026-06-30T02:34:52.729Z
交付范围：PC 管理端客户试用版

一、打开方式

1. 解压整个文件夹，不要只单独打开 index.html。
2. Mac 用户可在终端进入本文件夹后执行：
   python3 -m http.server 3010
3. Windows 用户可在本文件夹打开命令行后执行：
   py -m http.server 3010
4. 浏览器访问：http://127.0.0.1:3010

二、试用入口

- 首页左侧导航可查看完整产品模块。
- 点击“工单中心”或“开始客户试用”进入 PC 试用工作台。
- 本包不依赖 Vercel，适合客户本地电脑、内网服务器或国内云主机临时试用。

三、本次可试用范围

- 运营总览、资产台账、故障管理、工单中心、备件质量、设备履历、知识库等 PC 页面展示。
- PC 管理端客户试用工作台的流程演示。
- 详细路径见 docs/wind-smart-ops-customer-trial-manual.pdf。

四、暂不验收范围

- 手机端现场工程师真实登塔作业。
- 照片上传、电子签名、企业微信/钉钉/统一身份认证集成。
- 多风场、多组织、多用户压力测试。
